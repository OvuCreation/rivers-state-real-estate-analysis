# Rivers State Real Estate Market Analysis

## Project Overview

This project explores the real estate market across selected areas in Rivers State, Nigeria, using Python, SQL, and Power BI.

The objective is to identify pricing patterns, market trends, and investment opportunities within Port Harcourt and Obio-Akpor.

## Business Problem

Real estate prices vary significantly by location and property type. Investors, agents, and developers require data-driven insights to make informed decisions regarding property acquisition and development.

This analysis seeks to answer questions such as:

* Which neighborhoods are the most expensive?
* Which areas have the highest price per square meter?
* Which property categories dominate the market?
* What investment opportunities exist within the Rivers State property market?

## Dataset

The dataset consists of:

* 350 property listings
* 729 transaction records
* 12 neighborhoods within Rivers State

Areas covered include:

* GRA Phase 1
* GRA Phase 2
* Old GRA
* Rumuokoro
* Rumuodara
* Rumuola
* Woji
* Ada George
* Diobu
* D-Line
* Borokiri
* Town

## Python Analysis Performed

* Data inspection and cleaning

## Key Insights

* Premium pricing was concentrated within GRA neighborhoods.
* Price-per-square-meter analysis provided a more standardized measure of market value.
* Different property categories exhibited varying transaction volumes and values.
